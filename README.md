# Cricket Quiz App

This is my first frontend project built while learning HTML, CSS, and JavaScript.

The goal of this project was to understand:
- How HTML structures a webpage
- How CSS styles elements
- How JavaScript adds interactivity

## Features
- Multiple choice quiz
- Score calculation
- Result screen
- Play again functionality

## Technologies Used
- HTML
- CSS
- JavaScript

## What I Practiced
- DOM manipulation
- Styling layout with CSS
- Debugging errors
- Handling user interaction

## Purpose of This Project
- This project was created as part of my learning process to move toward becoming a frontend developer.
- More improvements and structured versions will be added as I continue learning.
